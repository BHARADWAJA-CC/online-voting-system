# Online Voting System

A simple web-based Online Voting System developed to understand frontend and backend integration.

## Features
- User vote submission
- One-vote-per-user validation
- Backend vote counting
- Result retrieval through API

## Tech Stack
- Frontend: React, HTML, CSS
- Backend: Python (Flask)

## Objective
To learn how a web application works by connecting frontend UI with backend APIs.

## Future Improvements
- Database integration
- User authentication
- Admin dashboard
- Secure deployment
